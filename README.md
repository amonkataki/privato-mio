# privato-mio

Repository pensata come "drive personale" privato.

## Obiettivo
Avere uno spazio con cartelle e file personali condivisibile solo con persone scelte da te.

## Accesso riservato
Per rispettare il requisito "accessibile solo da chi voglio":

1. Mantieni il repository **privato** su GitHub.
2. Condividi l'accesso aggiungendo solo i collaboratori che scegli (`Settings -> Collaborators and teams`).
3. Rimuovi i collaboratori quando non devono più accedere.

## Struttura consigliata
Puoi organizzare i contenuti in cartelle, ad esempio:

- `documenti-personali/`
- `documenti-condivisi/`
- `archivio/`

Ogni cartella può contenere sottocartelle e file in base alle tue esigenze.

## Condivisione occasionale
Quando vuoi condividere qualcosa:

1. carica/aggiorna i file nella cartella corretta;
2. consenti accesso al collaboratore necessario;
3. quando non serve più, revoca l'accesso.
